VertexArray¶

class ModernGL.VertexArray¶

Bases: ModernGL.ContextMember

release()¶

Release the vertex array.

render(mode, vertices, first, instances)¶

The render primitive (mode) must be the same as the input primitive of the GeometryShader.

Parameters:
  • mode (optional) – By default TRIANGLES will be used.
  • vertices (optional) – The number of vertices to transform.
Keyword Arguments:
 
  • first – The index of the first vertex to start with.
  • instances – The number of instances.
Returns:

None
transform(output, mode, vertices, first, instances)¶

Transform vertices. Stores the output in a single buffer. The transform primitive (mode) must be the same as the input primitive of the GeometryShader.

Parameters:
  • output – The buffer to store the output.
  • mode (optional) – By default TRIANGLES will be used.
  • vertices (optional) – The number of vertices to transform.
Keyword Arguments:
 
  • first – The index of the first vertex to start with.
  • instances – The number of instances.
Returns:

None
attributes¶
content¶

The content assinged to the VertexArray.

context¶

The Context object used for creation.

index_buffer¶

The index buffer if the index_buffer is set otherwise None.

program¶

The program assinged to the VertexArray. The program will be used automatically when rendering and transforming.

vertices¶

The number of vertices detected. This is the minimum of the number of vertices possible per Buffer. The size of the index_buffer will be used when the VertexArray has a valid index_buffer set. Per instance vertex attributes does not affect this number.
