Attribute¶

class ModernGL.Attribute¶

Bases: ModernGL.ProgramMember

array_length¶

If the attribute is an array the array_length is the length of the array otherwise it is 1.

location¶

The attribute location. The result of the glGetAttribLocation.

name¶

The attribute name. The name will be filtered to have no array syntax on it’s end. Attribute name without '[0]' ending if any.

program¶

The Program object used for creation.
